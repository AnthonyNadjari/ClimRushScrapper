Comment creer un token ?

  1. Allez sur github.com/settings/tokens
  2. Cliquez "Generate new token" → Fine-grained
  3. Selectionnez votre repo
  4. Permissions : Contents: Read & Write, Actions: Write
  5. Copiez le token et collez-le ici
